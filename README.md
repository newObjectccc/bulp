# bup

*bup is a scaffold for frontend enginer to help you setting your project. progressively and automatically integrate your project.*

## Installation

use npm:

```bash
npm install -g buildp
```

use pnpm:

```bash
pnpm add -g buildp
```

use yarn:

```bash
yarn add -g buildp
```

## Usage

for help:

```bash
bup -h
```

for eslint example:

```bash
bup eslint -f "yml"
```

for commitlint example:

```bash
bup commitlint"
```

for changelog example:

```bash
bup changelog -c"
```

- eslint integrate
