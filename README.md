# bup

_bup is a scaffold for frontend enginer to help you configure your project. progressively and automatically integrate development dependency of your project._

## Installation

use npm:

```bash
npm install -g buildp
```

use pnpm:

```bash
pnpm install -g buildp
```

use yarn:

```bash
yarn add -g buildp
```

## Usage

for help:

```bash
bup -h
```

for eslint:

```bash
bup eslint
```

for commitlint:

```bash
bup commitlint
```

for changelog:

```bash
bup changelog
```

for prettier:

```bash
bup prettier
```

for lint-staged:

```bash
bup lint-staged
```
